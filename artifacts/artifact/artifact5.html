Artifact 3: SchoolAI – AI-Assisted Learning Experience

A case study on deploying AI-powered personalized learning platforms to improve student engagement and outcomes.

Project Goals

  • Personalize learning paths based on student data
  • Provide timely feedback and support
  • Enable data-driven curriculum adjustments
  • Increase engagement through interactive AI tools

Technologies Used

  • Python (scikit-learn, TensorFlow), JavaScript (React.js)
  • Databases: PostgreSQL, MongoDB
  • Cloud: AWS for hosting and scalability

Sample Python Code Snippet

import pandas as pd
from surprise import Dataset, Reader, KNNBasic

# Load data
ratings_dict = {
    "itemID": [1, 1, 1, 2, 2, 3],
    "userID": [9, 32, 2, 45, 32, 9],
    "rating": [3, 2, 4, 5, 3, 4]
}
df = pd.DataFrame(ratings_dict)

reader = Reader(rating_scale=(1, 5))
data = Dataset.load_from_df(df[['userID', 'itemID', 'rating']], reader)

# Use KNNBasic algorithm
algo = KNNBasic()
trainset = data.build_full_trainset()
algo.fit(trainset)

# Predict rating for user 9 and item 2
pred = algo.predict(9, 2)
print(f"Predicted rating for user 9 on item 2: {pred.est}")

Scenario 1: Data Quality

  • Missing values, duplicates, outliers
  • Applied imputation, normalization, outlier removal

Sample Python Code

import pandas as pd
import numpy as np
from scipy import stats
from sklearn.preprocessing import MinMaxScaler

data = pd.read_csv('financial_data.csv')
data['income'].fillna(data['income'].mean(), inplace=True)
data['occupation'].fillna(data['occupation'].mode()[0], inplace=True)
data.drop_duplicates(inplace=True)
z_scores = np.abs(stats.zscore(data.select_dtypes(include=[np.number])))
data = data[(z_scores < 3).all(axis=1)]
scaler = MinMaxScaler()
data[['income', 'age']] = scaler.fit_transform(data[['income', 'age']])
