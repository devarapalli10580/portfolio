Artifact 1: AI Lab – No-Code Chatbot Assistant Development

This artifact documents my journey in developing an AI-powered chatbot assistant using no-code platforms to improve user engagement and automate routine queries.

Project Overview

The goal was to build a responsive, intuitive chatbot capable of handling common customer queries without traditional programming. Platforms used include Botpress, Chatbase, and Mizou AI.

Development Process

  1. Requirement Analysis: Defined chatbot objectives and scope.
  2. Platform Evaluation: Compared no-code tools for features, integration, and AI capabilities.
  3. Design: Created conversational flows and defined intents/entities.
  4. Training: Uploaded sample data and refined chatbot responses.
  5. Testing: Conducted user testing and improved accuracy iteratively.
  6. Deployment: Embedded chatbot on a demo website using iframe.

Technologies Used

  • Botpress – open-source chatbot platform
  • Chatbase – analytics for chatbot performance
  • Mizou AI – AI conversational design tool
  • Google Dialogflow – natural language understanding

Sample Intent JSON

{
  "intents": [
    {
      "name": "greeting",
      "examples": ["Hello", "Hi", "Good morning"],
      "responses": ["Hello! How can I assist you today?"]
    },
    {
      "name": "hours",
      "examples": ["What are your hours?", "When are you open?"],
      "responses": ["We are open Monday to Friday, 9 AM to 5 PM."]
    }
  ]
}

Challenges & Learnings

Understanding user intents precisely without coding required significant trial and error. The experience deepened my knowledge of conversational AI design and user experience principles.
