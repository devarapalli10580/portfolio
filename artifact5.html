Workshop 5 Portfolio Requirements

    • Title
    • Introduction and description
    • Objective
    • Process

Following these criteria ensures the portfolio is professional, informative, and valuable for prospective employers, peers, and instructors.

Relevance

Code Sample: Sample R Script Snippet

library(dplyr)
library(ggplot2)

# Load data
covid_data <- read.csv("covid_india.csv")

# Filter and summarize
summary <- covid_data %>%
  filter(Date >= "2021-01-01") %>%
  group_by(State) %>%
  summarise(
    total_deaths = sum(Deaths, na.rm=TRUE),
    total_vaccinations = sum(Vaccinations, na.rm=TRUE)
  )

# Plot deaths vs vaccinations
ggplot(summary, aes(x=total_vaccinations, y=total_deaths)) +
  geom_point(color="blue") +
  geom_smooth(method="lm") +
  theme_minimal() +
  labs(title="Deaths vs Vaccinations by State in India",
       x="Total Vaccinations",
       y="Total Deaths")

Code Sample: Python Weather Data Fetch

import requests

API_KEY = 'your_api_key_here'
city = 'Dallas'
url = f"http://api.openweathermap.org/data/2.5/weather?q={city}&appid={API_KEY}"

response = requests.get(url)
data = response.json()

print(f"Temperature in {city}: {data['main']['temp']} Kelvin")

Peer Feedback: Guidelines and Best Practices

Constructive peer feedback is essential for learning and improvement. Please follow these guidelines:

  • Start with Positive Observations: Acknowledge what the peer has done well.
  • Be Specific: Provide clear examples rather than vague comments.
  • Suggest Improvements: Offer actionable advice on how to enhance the portfolio or artifacts.
  • Focus on Work, Not the Person: Keep feedback professional and objective.

Areas to Focus on

  • Clarity of project descriptions and objectives.
  • Demonstration of AI/ML skills and data analytics.
  • Overall organization, design, and presentation of the portfolio.
  • Appropriateness and relevance for the intended audience.

Confidentiality & Professionalism

Respect the intellectual property and privacy of peers’ work. Provide feedback as you would in a professional setting.
