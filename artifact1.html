Artifact Details

Overview

The featured artifact is a conversational AI assistant – the ChatBox Creator Bot – designed to empower non-programmers to create and deploy custom chatbots without coding. It was built using Mizou.com, a no-code platform allowing easy prototyping and deployment of chatbot flows.

This chatbot functions as a mentor, guiding users through tone selection, FAQ design, visual flow planning, and integration into web platforms. It primarily serves beginners and small business owners eager to adopt conversational AI.

Problem Solved

Many innovative individuals face steep learning curves when implementing chatbots due to technical barriers. This AI assistant simplifies the chatbot creation process via intuitive conversational interactions, visual aids, examples, and templates.

Design Thinking Process & Steps

Conducted interviews with peers, entrepreneurs, and small business owners to understand challenges in chatbot development. Primary barriers included technical jargon, interface complexity, and unclear use cases.

Framed the challenge: How might we help non-coders build functional chatbots quickly and confidently without writing code?

Brainstormed over a dozen features including interactive FAQ builder, tone selector, visual flowchart previews, multilingual templates, and user success checklist.

Used Mizou to implement core chatbot features with embedded documentation and designed a clean, intuitive user interface resembling teaching interactions.

Conducted peer testing sessions; revised language style, added memory features for session continuity, and integrated visual aids. The final Mean Reciprocal Rank (MRR) performance improved to 0.86.

Challenges & Solutions

  • Challenge: Users found initial language too complex.
    Solution: Simplified chatbot language to 8th-grade reading level with examples and analogies.
  • Challenge: Linear conversation flows confused users.
    Solution: Added inline flowcharts and multimedia tutorials.
  • Challenge: Onboarding was unclear.
    Solution: Implemented a “Welcome & Instructions” sequence for new users.

Reflection

This project strengthened my user-centric design skills by emphasizing simplification of complex AI concepts into accessible interfaces. Collaborating with peers from diverse backgrounds enhanced my communication and teamwork abilities.

Iterative feedback was invaluable for refining chatbot tone, interaction flow, and visual aids, resulting in a polished final product. This artifact exemplifies how empathy, design thinking, and AI expertise come together to solve real problems.
