Artifact Description

Overview: The featured artifact is a conversational AI assistant—ChatBox Creator Bot—built using Mizou.com. It helps non-technical users create and deploy chatbots without coding skills.

Problem Solved: Simplifies chatbot development for small businesses and non-programmers by providing an intuitive, guided process.

Tools Used: Mizou.com | Prompt Engineering | Design Thinking

Design Thinking Applied

Empathize: Conducted interviews with small business owners to identify barriers in chatbot adoption.

Define: How might we help non-coders build functional chatbots quickly and confidently without writing code?

Ideate: Brainstormed features like tone selector, interactive FAQ builder, and visual flow previews.

Prototype: Implemented a no-code solution using Mizou with embedded visual aids.

Test: Improved performance based on user feedback. Achieved MRR of 0.86.
