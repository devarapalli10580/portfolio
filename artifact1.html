Artifact 1: Eco-Coach Chatbot Project

Conversational AI for Sustainable Living

Project Overview

  • Type: Conversational AI Application
  • Tools: Botpress, Dialogflow, Python
  • Date: July 10, 2025

The Eco-Coach Chatbot is an AI-driven assistant that promotes sustainable practices by providing eco-friendly tips, answering questions on energy conservation, and integrating APIs for real-time updates. It was designed to encourage environmental awareness through an engaging and user-friendly interface.

Key Features

  • Uses NLP for natural, human-like conversations.
  • Provides personalized eco-friendly tips based on user preferences.
  • Integrates external APIs for real-time sustainability updates.

Skills Demonstrated

Skill Description
AI/ML Implementation Developed chatbot with NLP and intent recognition.
Programming Integrated Python scripts for handling API calls and responses.
UX Design Designed intuitive conversation flows for seamless user experience.

Challenges & Solutions

Challenge: Handling ambiguous user queries.

Solution: Implemented context-based fallback intents and trained the model with diverse scenarios.

Reflection

This project allowed me to explore the intersection of AI technology and sustainability. It enhanced my understanding of natural language processing, real-time API integration, and user engagement strategies.
