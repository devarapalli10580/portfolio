Introduction

Hello, my name is Naga Venkata Satyanarayana Devarapalli, and I am pursuing a Master of Science in Artificial Intelligence – Data Analytics at Indiana Wesleyan University.

This portfolio showcases real-world applications of AI/ML, reflecting my journey and technical expertise. The featured artifact originates from Workshop 1, where I designed and tested an AI assistant chatbot using design thinking principles.

Artifact Description

The artifact, ChatBox Creator Bot, helps users create and deploy chatbots without coding, using Mizou.com. It supports features like tone selection, FAQ design, and flowchart previews.

Assignment Objectives

  • Practice Design Thinking methodology in AI projects.
  • Learn prompt engineering and user-centric chatbot design.
  • Implement no-code AI solutions using real-world platforms.

Development Steps

Key steps in developing the artifact:

  • Conducted user interviews to identify challenges.
  • Brainstormed features like multilingual templates, tone selector, and visual previews.
  • Prototyped chatbot flows on Mizou.com and performed peer testing.
  • Refined tone and user experience based on feedback.

Reflection

This project enhanced my skills in user empathy, interdisciplinary collaboration, and iterative design. User testing revealed key improvements, such as simplified language and onboarding sequences.
